## Problems Encountered & Solutions

### Problem 1

#### Description 
- Extended time required for feature implementation.

#### Root Cause
- The team lacked the specific technical skills needed.

#### Solution Applied 
- Redistributed workload by reducing other responsibilities for the primary developer.

### Problem 2

#### Description
- the lack of first steps to implementation in issues.

#### Root Cause
- Issues were created based on user stories, but no user stories existed for the initial steps.

#### Solution Applied
- Conducted a grooming session and created  issues for all initial steps.
